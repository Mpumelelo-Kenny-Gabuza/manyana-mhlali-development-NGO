# manyana-mhlali-development-NGO
This is the official website for Manyana Mhlali Development (MMD), a registered Non-Profit Company (NPC) based in Libode, Eastern Cape, South Africa. The website serves as a digital platform to showcase the organization's programmes, build community trust, facilitate donations, and provide resources for those seeking support

Website Purpose
The website aims to:

- Educate the public about MMD's mission and programmes

- Build trust with donors, partners, and the community

- Facilitate donations through multiple secure channels

- Provide support resources for community members

- Showcase impact through stories and statistics

- Ensure transparency through governance information
